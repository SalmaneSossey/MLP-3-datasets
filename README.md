# MLP-3-datasets

Starter structure for MLP experiments across 3 datasets.

## Layout

- `notebooks/TP2_MLP_3datasets.ipynb`
- `src/seed.py`
- `src/tabular.py`
- `src/image_tabular.py`
- `src/models.py`
- `src/train.py`
- `src/metrics.py`
- `requirements.txt`

## Quick start

```bash
pip install -r requirements.txt
python src/train.py
```
